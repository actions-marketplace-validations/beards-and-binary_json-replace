# JSON Value Replacer

This action updates the passed JSON using the passed properties and values.

## Inputs

### `json`
**Required** The JSON to update

### `updates`
**Required** The list of Updates to process, provided as key / value pairs
Example: `[['key1', 'value 1'], ['key2', {id: 1}]]

## Outputs
### `json`
The updated JSON
